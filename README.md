# SpotifyToVK
### Display your current listen to a song from Spotify in the VK status.
![1.1](img/spotify.png)

# Installation

###### Clone repository
```bash
$ git clone https://github.com/xcaq/spotify-to-vk.git
```

###### Go to "spotify-to-vk" folder
```bash
$ cd spotify-to-vk
```

###### Install libraries using pip

```bash
$ pip3 install -r requirements.txt
```

# Setting up:

###### Config
![Version 1.0 ](img/config.png)
###### For easily settings you can use the command below, or use links in the FAQ by yourself. 

```bash
$ python3 first.py
```

# Usage:

```bash
$ python3 main.py
```
